Detailed Description

template<class T>
class orderedList< T >

a list of items that maintains the order in which items are added or removed.

The orderedList class uses a linked list of the class orderedListItem to keep track of when items are added to the beginning (shift()) or end (push()) of the list, or when items are removed from the beginning (unshift()) or end (pop()) of the list.
